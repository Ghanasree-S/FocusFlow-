ğŸ§  Understanding the ML Models

We use three different models because each captures different patterns in your data. Think of it like asking three experts for their opinion and averaging their answers!

1. LSTM (Long Short-Term Memory)

Deep Learning Neural Network

What it is: A type of neural network that can "remember" patterns from the past. It's like having a really good memory!

What it learns:

  • If you worked hard Mon-Tue, you might slow down Wed
  • After a vacation, productivity takes time to recover
  • Complex patterns that aren't obvious to humans

How it works (simplified):

# LSTM looks at last 7 days to predict tomorrow input = [Mon, Tue, Wed, Thu, Fri, Sat, Sun] # past week output = predict(input) # tomorrow's productivity

Why we use it: Best at finding complex, non-linear patterns

2. ARIMA (AutoRegressive Integrated Moving Average)

Statistical Classical ML

What it is: A classic statistical model used since the 1970s. It's the "gold standard" for time series forecasting.

Our configuration: ARIMA(1, 0, 0)

  • p=1: Use yesterday's value to predict today (AR part)
  • d=0: Data is already stable, no trend removal needed
  • q=0: Don't use past errors for prediction

The formula:

Today's Productivity = c + (0.xx Ã— Yesterday's Productivity) + error # Example: # If yesterday = 200 min and coefficient = 0.8 # Today â‰ˆ 30 + (0.8 Ã— 200) = 190 min

Why we use it: Simple, interpretable, great for trends

3. Prophet (by Facebook/Meta)

Seasonality Additive Model

What it is: A model developed by Facebook specifically for business time series with repeating patterns.

What it learns:

  • Mondays are usually more productive than Fridays
  • Weekends have 40% less productivity
  • Monthly patterns (start of month vs end)

The formula:

Prediction = Trend + Weekly_Pattern + Holidays + Error # Example for a Friday: # Trend: 200 min (base level) # Weekly: -30 min (Fridays are lower) # Friday prediction: 170 min

Why we use it: Automatically finds day-of-week patterns

Why Combine All Three? (Ensemble)

Each model has strengths and weaknesses. By combining them, we get the best of all worlds!

Final Prediction = (0.4 Ã— LSTM) + (0.3 Ã— ARIMA) + (0.3 Ã— Prophet) # Example: # LSTM predicts: 180 min # ARIMA predicts: 200 min # Prophet predicts: 170 min # Ensemble = (0.4 Ã— 180) + (0.3 Ã— 200) + (0.3 Ã— 170) # = 72 + 60 + 51 # = 183 min (final prediction)
Model Weight Reason
LSTM 40% Best at complex patterns
ARIMA 30% Most accurate (lowest error)
Prophet 30% Best at weekly patterns
